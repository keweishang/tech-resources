# Gem

List all gems installed locally:

```
gem list --local
```

Install gem (latest version):

```
gem install jekyll
```

Install gem with target version:

```
gem install jekyll -v 3.8.5
```

Uninstall gem:

```
gem uninstall i18n
```

Uninstall gem with target version:

```
gem uninstall i18n -v 1.6.0
```
